# Desktop Pet · 桌宠项目 — 功能说明

> 同人桌宠项目，与 DeepSeek 或任何官方品牌**无关**。  
> 启动：双击 `启动宠物.bat` 或 Release 里的 `DesktopPet.exe`  
> 技术栈：Python + PyQt6 · 透明置顶 · 存档 `%APPDATA%\DesktopPet\`（Windows）

---

## 她是谁？

这是一只住在桌面上的 **桌宠**（蓝白主题角色），会待机、走路、睡觉、被撸、被喂食，偶尔还会冒气泡跟你说话。

你打工赚钱、她陪你看棋、陪你掷骰——  
**你是铲屎官，她是你的桌面搭档。**

---

## 一、桌宠本体 · 日常陪伴

### 窗口与存在感

- **透明无边框**小窗，始终置顶，不挡任务栏逻辑（可多屏、记住位置）
- **系统托盘**常驻：左键显示/隐藏，右键管理宠物
- 支持 **多只宠物** 同时存在（右键「添加新宠物」）
- 关闭窗口 = 藏起来，不是退出；真要告别请点「退出程序」

### 动画与动作

皮肤在 `skins/default/` 下，按文件夹分动作（idle / walk / sleep / eat / happy / sad ……）。

| 你做什么 | 她怎么反应 |
|---------|-----------|
| 单击 | 冒对话气泡，播 click 类反馈 |
| 拖拽 | grabbed → 松手 fall → 回到 idle |
| 悬停 | 有 hover 皮则播放 |
| 久不互动 | 可能自己 **walk** 溜达 |
| 更久不动 | 进入 **sleep**（Zzz，帧率已调顺） |
| 饿/心情低 | **sad** 或定时喊饿气泡 |

动作由 **ActionManager + actions.json** 调度，含眨眼 overlay、走路速度、睡眠 ping-pong 等参数。

### 三维状态（她是活的）

- **饱食度**：随时间下降，太低会 sad / 喊饿
- **心情值**：随时间下降，互动可恢复
- **好感度**：喂食、抚摸、玩耍累计；有 **等级曲线**（越往后升级越慢）

互动有 **防刷限速**（每分钟喂食/抚摸/好感上限），防止一键刷满。

### 对话气泡

- 头顶圆角气泡，按场景抽不同 **文本池**（普通 / 开心 / 饥饿 / 低心情 / 游戏胜利等）
- 支持主题色、accent 样式（`BUBBLE_THEME`）
- 玩小游戏赢了、今日金币打满，也会冒专属台词

### 右键菜单（对她点右键）

- **喂食** — 消耗背包里的汉堡，加饱食 & 心情（没货会提示去商店）
- **抚摸** — 加心情，可能 happy + 爱心气泡
- **唤醒** — 睡着时点一下叫醒
- **切换皮肤** — 扫描 `skins/` 目录
- **显示状态栏** — 头顶小 HUD：饱食 / 心情 / 好感等级
- **小商店 / 背包** — 经济系统入口
- **陪玩打工** — 小游戏大厅（见下文）
- **设置 / 隐藏 / 退出**

开发者模式下还有 **滑翔调试** 菜单（将 `src/utils/constants.py` 里 `DEV_MODE = True` 后可见）。

## 二、小商店 & 背包 · 养得起她

- 初始金币（开发模式会慷慨很多）
- **汉堡** 🍔：商店购买 → 背包囤积 → 右键喂食消耗
- 金币、背包、每日打工上限 **自动存档**
- **每日金币上限** 250（防止无限刷，满了会气泡吐槽）

## 三、陪玩打工 · 小游戏大厅

右键 **陪玩打工** 打开大厅。  
玩任何游戏时，桌宠会进入 **旁观模式**（挪到屏幕边上看你玩，不挡棋盘）。

### 旗舰对局（大窗口、独立会话）

#### 🎲 大富翁 · 中国城市巡回赛

- **2～4 人**：1 个本地玩家 + 最多 3 电脑，或自定义人机混合
- **开局可选头像**（预设 + 本地图片）
- **24 格环形棋盘**，中国城市地名 + 机会 / 命运 / 税 / 监狱 / 进监狱 / 免费停车
- **经典方形 2.5D 棋盘**：滚轮缩放、左键拖移；棋子 **逐格跳跃** 动画（已修重影、回弹、黑屏等问题）
- **快节奏经济**：高起始金、高租金、过起点发工资；约 15～25 分钟一局
- **购地 / 收租 / 盖房 / 垄断双倍租金**
- **机会 & 命运卡**（进监狱、后退、领钱、维修费……）
- **监狱**：踩「进监狱」或抽卡入狱 → 下回合 **停一回合**（可交保释金立即掷骰）；路过「监狱格」不算入狱
- **Kenney CC0 音效 + BGM**，棋盘粒子（掷骰、买地、收租、胜利彩带等）
- **人类获胜** 有金币奖励 + 桌宠开心反馈

#### ♟ 国际象棋

- **人机对战**（选执白/执黑，简单 / 普通 / 困难）
- **双人同屏** 本地轮流
- 经典 **8×8 绿格棋盘**，Unicode 棋子（无版权图包）
- 点选走棋、升变弹窗、将军高亮、认输
- 电脑 **后台算棋**，界面不卡死
- 将死获胜同样有金币奖励

### 轻量小游戏（窗口内嵌、即开即玩）

| 游戏 | 玩法概要 | 操作 |
|-----|---------|------|
| 🐍 贪吃蛇 | 吃汉堡变长赚分 | 方向键 / WASD |
| 🍔 接汉堡 | 60 秒内接食物躲炸弹 | ← → |
| ☄️ 流星躲避 | 存活越久分越高 | 方向键 |
| 🧠 美食记忆 | 90 秒内翻牌配对 | 鼠标 |

得分换算金币，受 **每日上限** 约束。

## 四、数据与设置

- **自动保存**：宠物位置、皮肤、三维状态、金币、背包、游戏战绩
- **设置面板**（托盘或右键 → Settings）：**18 种语言**、自动溜达、状态栏 HUD、窗口透明度
- 皮肤管线：`rebuild-skins.bat` / `重建皮肤.bat` / `tools/build_skins.py` 从本地序列图生成 `skins/default/`
- GPU 抠图工具在 `tools/dev/`（本地开发用，未随 Release 分发）

## 六、已知边界 & 还没做的（诚实清单）

- 大富翁 **续局存档**、3D 骰子物理、联机 — 未做
- 国际象棋 **无悔棋、无棋谱导出**；困难 AI 仍属「入门级」
- 桌宠 **暂无语音、暂无 LLM 实时对话**（气泡是本地文本池）
- 默认 `DEV_MODE = False`（正式金币 80 起）；开发调试可改回 `True`
