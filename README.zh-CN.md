## 从源码运行

```powershell
git clone https://github.com/zfhooyfincasnpktlmeisxv-cpu/DesktopPet.git
cd DesktopPet
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python src\main.py
```

或双击 **`启动宠物.bat`** / `start.bat`。
