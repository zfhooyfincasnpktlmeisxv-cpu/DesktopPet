# Changelog

All notable changes to this project are documented here.

Format based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-06-16

### Added

- Desktop pet with idle/walk animations, speech bubbles, hunger/mood stats
- Shop economy, backpack, and daily gold cap
- Mini-games: Snake, catch burgers, meteor dodge, memory match
- Monopoly board (24 tiles, Chinese cities) with SFX
- Chess vs AI or local two-player
- 18 UI languages (English default)
- Windows standalone `DesktopPet.exe` and Inno Setup installer
- GitHub Actions CI test workflow

### Fixed

- README screenshot mapping for chess and arcade sections
- Installer wizard top-right icon black-edge artifact
