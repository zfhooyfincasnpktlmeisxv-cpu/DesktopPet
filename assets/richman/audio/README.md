# 大富翁音频（CC0）

首次打开对局会自动从网络拉取；也可手动运行：

```bash
python scripts/download_richman_audio.py
```

## 来源

| 类型 | 来源 | 许可 |
|------|------|------|
| 骰子 / 筹码 / 卡牌音效 | [Kenney Casino Audio](https://kenney.nl/assets/casino-audio) | CC0 |
| 胜利短乐 | [Kenney Music Jingles](https://kenney.nl/assets/music-jingles) | CC0 |
| 对局 BGM | OpenGameArt *four_loop* | 见 OGA 页面 |

## 目录

- `sfx/` — 游戏事件音效（`.ogg`）
- `bgm/game.mp3` — 循环背景音乐
- `_src/` — 原始 zip 缓存（可删，会重新下载）
